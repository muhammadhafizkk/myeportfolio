Reflections for each assessment

For Assignment 1: Report, I used Google Lighthouse to compare and assess five different e-commerce websites. This assignment taught me how to assess important website features like speed and accessibility. I can pinpoint important advantages and disadvantages in the functionality and web design of websites by contrasting them side by side. The developers can use these data as a roadmap to boost the overall efficiency of their web platforms.
More visual aids, such as graphs and charts, might be added to the study to improve its impact and clarity, in my opinion.

In Assignment 2: Website, I used HTML and CSS Grid layout to develop a landing page for my personal website. Through this assignment, I was able to learn how to logically structure the webpage into sections that are easy to navigate, such as the header, left and right menus, content, and footer. They assisted with the replication of a polished and visually appealing design, complete with a copyright notice, navigation menu, and welcome header, using the theme examples that were provided as references. In order to guarantee both the aesthetic and cross-device responsiveness, CSS was crucial.
All things considered, this assignment not only strengthened my grasp of HTML and CSS Grid but also emphasized how crucial user-centered design and personalization are to web development.

This assignment, Assignment 3: GitHub e-Portfolio, required me to use the GitHub platform to develop an e-portfolio that included web programming. This task was a rewarding experience that gives me the opportunity to present my academic background in a structured, digital style. It also evaluated my general knowledge of and skill with the HTML, CSS, and JavaScript languages. Creating the e-portfolio on GitHub helped me to better understand repository management and version control. Seeing all of my work on one unified platform was satisfying. This not only made it easier for me to evaluate my achievements but also pointed me areas where I may still improve.

As part of the Mini Project, we created a web-based mechanism for lodging damage complaints. Participating in the project gave me hands-on experience using my theoretical knowledge to solve practical issues. It put to the test every bit of our knowledge of HTML, CSS, JavaScript, PHP, MySQL, and local website hosting with help from GitHub and localhost000. It improved my problem-solving abilities, encouraged creativity and invention, and improved my teamwork abilities including cooperation and communication.
Since the specified software might occasionally be changed, I propose that the website hosting instruction be updated and expanded upon.
